## 配图使用建议

### 优先级

| 优先级 | 故事 | 理由 |
|--------|------|------|
| P0 优先出图 | 1(全息碎片)、9(掌骨穴位)、15(耳穴对比) | 视觉冲击力最强，一眼理解概念 |
| P1 第二批 | 6(侏儒图)、7(皮肤地图)、17(眼睛到脑) | 辅助理解难懂概念 |
| P2 第三批 | 剩余 11 个 | 锦上添花 |

### 使用方式

1. **AI 生成**：将 Prompt 复制到 Midjourney / DALL-E / Stable Diffusion 生成
2. **素材替换**：用搜索关键词找 CC0/CC-BY 许可图片（Wikimedia Commons、Unsplash）
3. **文件命名**：`images/story-01-hologram.jpg`、`images/story-09-metacarpal.jpg`
4. **README 引用**：`![全息碎片](images/story-01-hologram.jpg)`

### 风格统一建议

- 所有配图使用同一种视觉风格（推荐：科学插图风格，非真实照片）
- 色系统一：蓝-紫-金色调（与全息概念呼应）
- 尺寸统一：16:9 或 4:3
- 每张图底部加一行图注（caption）
