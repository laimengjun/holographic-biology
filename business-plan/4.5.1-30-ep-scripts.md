## 第 1 集 · 你身体里有 5 个"完整的人"

**【Hook · 0:00-0:10】**
（特写：一只耳朵的微距镜头，慢慢放大）
你看到的这只耳朵，如果放大 1000 倍，它像一个蜷缩的婴儿。耳朵上有 200 多个点位，据说对应你全身的器官。这是真的吗？这就是"全息"。

**【上集回顾 · 0:10-0:30】**
这是 30 集课程的第一集。30 集我会讲完张颖清从 1972 到今天的 50 年——一个中国本土学派如何诞生、争议、被遗忘，又在 AI 时代以另一种方式复活。

**【主体 · 0:30-8:00】**

**一、最直觉的"全息"——耳朵**
1972 年，一个山东年轻人盯着一张耳朵图，发现耳朵的形状像一个倒置的胎儿。耳朵顶端对应头部，下面是对应的脊椎、四肢、内脏。如果这是真的，按耳朵某个点，就能治身体的某个部位。

**二、5 个"完整的人"**
这个年轻人叫张颖清。他说人体至少有 5 个这样的"微系统"——耳朵、第二掌骨（就是手掌那根）、足底、面部、头皮。每个微系统都像一个缩小的人，包含了全身的信息。这 5 个，就是全息律最直觉的版本。

**三、但是这 5 个真的是"完整"的吗？**
（实验镜头：医生按患者手掌第二掌骨 5 分钟）
医学界对这些"微系统"的态度很分裂。支持派说有效，反对派说证据不足。今天，第 1 集先不争论，我们后面会用 19 集去拆。

**【AI 映射 · 8:00-9:00】**
有意思的是，AI 时代有个非常类似的概念叫"模块化设计"——把一个完整系统装进一个小模块。亚马逊的 Alexa、智能音箱里的语音识别，都是把"听声音→理解意图→回答"这套完整链路装进一个小盒子。耳朵是身体的 Alexa 吗？

**【金句 · 9:00-9:30】**
"如果你能用耳朵治病，那你的脚底为什么不能？这是张颖清 50 年前问的问题，今天我们仍在追问。"

**【CTA · 9:30-10:00】**
下集我们去看 1973 年的张颖清——一个没读完大学的赤脚医生，怎么发现了"第二掌骨线"。订阅 + 评论"全息"，我会在 30 集里和你一起实验验证韦三立的金边虎皮兰。

**视觉资产清单**：

## 第 2 集 · 1973 年，一个赤脚医生发现了一条线

**【Hook · 0:00-0:10】**
（特写：一只手的手掌，一根第二掌骨被红笔标出）
这条线，就是张颖清一生事业的起点。1973 年他注意到，第二掌骨（就是你手掌最靠近大拇指的那根骨头）的不同位置，对应全身不同部位。这个发现后来被称为"第二掌骨节肢系统"。

**【上集回顾 · 0:10-0:30】**
上集我们讲了耳朵——5 个"完整的人"中的一个。这集我们要回到 1973 年，看看张颖清是怎么发现"第二掌骨"的。

**【主体 · 0:30-8:00】**

**一、谁是张颖清**
1947 年生于内蒙古包头。没读完大学，在山东做赤脚医生。1972 年开始研究手和耳朵的对应关系。1 年后，他发现第二掌骨——手掌最靠近大拇指的那根骨头——上面有一组点位，对应全身。

**二、第二掌骨是什么？**
（动画：手掌放大，第二掌骨高亮）
第二掌骨节肢系统的关键发现是：这根骨头上有 5 个核心点位——头/颈/上肢/肺/心/肝/胃/肾/下腹/足。如果你胃痛，按第二掌骨靠手腕的那个点，可能有用。

**三、一个赤脚医生的"诺贝尔梦"**
1972 年到 1981 年，张颖清用 9 年时间，把这个发现写成论文。1981 年发表在《自然杂志》第 4 期——不是顶级期刊，但他是自己投稿的。2 年后第一次全国生物全息研讨会在内蒙古集宁召开，1987 年《生物全息诊疗法》出版，发行 7.5 万册。

**四、为什么学界没当回事**
（动画：主流期刊 vs 张颖清的发表路径）
主流生物学界没当回事。原因：他跳过了同行评议关键步骤——没有用大样本临床试验验证。他后来 1995 年的论战就因为这个。

**【AI 映射 · 8:00-9:00】**
有意思的是，独立开发者、AI 极客圈里也有类似故事。Stable Diffusion 出来前，作者从学术界被排挤，最后在大公司开源。这两个故事的共同点：**主流不认的东西，可能在另一个世界是颠覆性的**。

**【金句 · 9:00-9:30】**
"一个赤脚医生在山东看手掌，一个研究员在硅谷做神经网络。50 年后他们发现，结构竟然这么像。"

**【CTA · 9:30-10:00】**
下集我们做实验——把金边虎皮兰砍成 6 段，看看会发生什么。这是张颖清学派唯一一个能在家复现的实验。

**视觉资产清单**：
1. 张颖清 1973 / 1987 照片
2. 第二掌骨节肢系统详细图
3. 1981 年《自然杂志》原页

## 第 3 集 · 把植物砍成 6 段，每段都长成新植物

**【Hook · 0:00-0:10】**
（特写：一片金边虎皮兰叶子，被切成 6 段，每段放入土中）
金边虎皮兰——一种叶片边缘金黄色、中间绿色的植物。1981 年，一个叫韦三立的人把它砍成 6 段。9/10 长成了新植物。这件事，是张颖清全息生物学里**唯一**一个你回家能复现的实验。

**【上集回顾 · 0:10-0:30】**
上集我们看了张颖清怎么发现第二掌骨。这集我们看他的同事韦三立——一个在北京农大读研究生的年轻人。

**【主体 · 0:30-8:00】**

**一、韦三立是谁**
1954 年生，比张颖清小 7 岁。1981 年在北京农大读研究生，专攻植物繁殖。1981 年开始做这个实验——用张颖清的理论预测叶片不同位置长出来的植物，应该有不同特征。

**二、实验设计**
（动画：实验步骤）
- 取金边虎皮兰叶片横切成 6cm 段
- 9/10 成活（金边消失）
- 来自叶片**中部**绿色部分的新植株：全绿色 + 褐色虎纹
- 来自叶片**边缘**黄色部分的新植株：**全黄色无虎纹**

**三、为什么这很重要？**
传统植物学认为，营养繁殖应该保留亲代特征。韦三立的实验显示：**繁殖结果取决于该部分在整体中的位置**——这支持张颖清"部分与整体同构"的预测。

**四、张颖清的反应**
1983 年第一次全国生物全息研讨会上，韦三立把实验材料提交给了张颖清。张颖清当时**赞扬**了韦的工作。但——张颖清**没有重复**该实验，**也没有做分子生物学层面的验证**，而是直接把结果上升为"高活性基因组合理论"。

**五、遗憾**
王德奎在 F-02 里批评这种"从具体实验直接跳到宏大理论"的跳跃。韦三立的实验至今没有被任何独立第三方系统重复。

**【AI 映射 · 8:00-9:00】**
这个故事对应 AI 研究里的一个老问题：**单次实验 vs 重复验证**。OpenAI 的 GPT-4 出过很多"涌现能力"的报告，但 Anthropic 试图重复时，很多无法复现。这就是科学严谨性的代价——慢，但是真。

**【金句 · 9:00-9:30】**
"学派里最该被记住的不是张颖清的理论，而是韦三立的实验方法。"

**【CTA · 9:30-10:00】**
这周你也可以试试——找一片多肉或绿萝的叶子，切成几段，看不同位置的叶子是否长出不同特征。把照片发评论区，我们做云端实验。

**视觉资产清单**：
1. 金边虎皮兰不同段长出的新植株对比
2. 韦三立 1981 年实验原始记录
3. 张颖清 1983 年会议照片

## 第 4 集 · 全息 vs Hologram——你可能搞混了 50 年

**【Hook · 0:00-0:10】**
（动画：左边是物理学的全息图，右边是张颖清的"全息耳"）
左边是物理学的全息图（Hologram）——用激光记录光的干涉图案，能从一张 2D 底片重建 3D 图像。右边是张颖清的"全息耳"——耳朵包含全身信息。这两个"全息"，根本不是一回事。

**【上集回顾 · 0:10-0:30】**
上集我们看了金边虎皮兰实验。今天要解决一个根本误解——张颖清用的"全息"和物理学的"全息"，名字一样，意思完全不同。

**【主体 · 0:30-8:00】**

**一、5 个"全息"概念**
（动画：5 个方向对比图）
1. **A Holobiont**（1990s+）—生物体 + 共生微生物的总和
2. **B Morphic Resonance**（1981 Sheldrake）—形态场，跨代传递
3. **C Holographic Brain**（1971 Pribram）—大脑存储是全息式的
4. **D Quantum Holography**（1993 Goswami）—量子层面的全息
5. **E Fractal Organism**（1982 Mandelbrot）—分形生物体
6. **F 张颖清全息生物学**（1947-2004 中国）—我们的主方向

**二、张颖清的"全息"是什么意思**
张颖清的"全息"= 部分包含整体信息。这跟物理 hologram 的"光波干涉记录"机制**没有直接关系**。他 1989 年干脆把"全息胚"改名为 ECIWO（Embryo Containing the Information of the Whole Organism），强调这是**生物学**而非物理学意义上的全息。

**三、为什么他要"借词"**
（史料照片：张颖清会议照片）
物理学 hologram 在 1980 年代非常火（光全息术是热门话题）。张颖清用"全息"这个词能让国外读者产生联想——"哦，是那个高级的物理概念"——从而提高传播效果。这是修辞策略。

**四、但也付出了代价**
学界批评："你借物理学的术语又不解释清楚，会让人误解"。这种术语借用是国际传播的便利，也是**科学严谨性的妥协**。

**【AI 映射 · 8:00-9:00】**
AI 圈也有类似的"借词"现象。比如 "transformer"（变压器）"embedding"（嵌入）"attention"（注意力）—— 这些数学/物理概念被借用，含义已经完全不同。这是跨学科传播的代价。

**【金句 · 9:00-9:30】**
"张颖清自己都知道，他用的'全息'是借词。50 年后我们回头看，他没说错，但说得不清。"

**【CTA · 9:30-10:00】**
下集我们看完整 50 年时间线——3 个时代，1 个学派。

**视觉资产清单**：
1. 5 方向对比图（来自 concept-map.md）
2. 物理学 hologram 演示
3. 1989 年 ECIWO 改名公告

# 第二部分 · 历史脉络（第 6-12 集）

## 第 6 集 · 1981，那篇改变张颖清一生的论文

**【Hook · 0:00-0:10】**
（特写：一本 1981 年的《自然杂志》第 4 期）
1981 年，张颖清在《自然杂志》第 4 期发表《生物全息律》。这是他的代表作，也是他被后来批评者反复拿来说事的起点。

**【上集回顾 · 0:10-0:30】**
上集我们看了 50 年时间线。这集聚焦第一个关键节点——1981 年的论文。

**【主体 · 0:30-8:00】**

**一、《生物全息律》说了什么**
论文核心命题：**生物体的每一相对独立的部分（如耳、第二掌骨、足底）是整体成比例的缩小，包含了整体各部位的全部信息；通过刺激这些"全息胚"上对应的位点，可以诊断和治疗整体对应部位的疾病。**

**二、为什么发表在《自然杂志》而不是更高级的期刊**
（史料：1981 年中国学术期刊生态）
《自然杂志》是中国国内期刊，不是英国 Nature。1981 年中国学术期刊分级制度还在建设中，张颖清选择这里——可能是渠道问题，可能是审查问题，也可能是他没意识到发顶刊的重要性。

**三、论文引发的国内反响**
（动画：反应涟漪）
- 1982：《生物体结构三定律》出版
- 1983-09：第一次全国生物全息研讨会（内蒙古集宁）
- 1985：《生物全息律》专书出版
- 1987：《生物全息诊疗法》发行 7.5 万册

反响不算小。但主流生物学界没动静——没有人在《中国科学》或《科学通报》上写评论。

**四、隐藏问题：没做"同行评议"**
（动画：传统期刊评议流程 vs 张颖清路径）
张颖清的论文**没有走完整的同行评议流程**。他跳过了"被同领域专家先审核"这一步。这在 1980 年代的中国学界是可能的（当时规则没那么严），但在国际主流期刊是不可能的。

**【AI 映射 · 8:00-9:00】**
arXiv 的开放发表也有类似争议——预印本未经审阅，作者可以快速发布，但同行评议仍是科学质量的护城河。AI 圈的"Llama 论文被社区复现成功"才算真的"发表"——这是新时代的"评议"。

**【金句 · 9:00-9:30】**
"1981 年张颖清发了一篇论文，世界没有停下。但对张颖清来说，世界从此开始。"

**【CTA · 9:30-10:00】**

## 第 7 集 · 5 部专著 vs 主流学界——他用书堆出一个学派

**【Hook · 0:00-0:10】**
（特写：5 本书封面依次出现）
从 1982 到 1992，10 年 5 本书：《生物体结构三定律》《生物全息律》《生物全息诊疗法》《全息生物学》《全息胚及其医学应用》。这不是学者，是出版机器。

**【上集回顾 · 0:10-0:30】**
上集我们看了 1981 年那篇论文。这集看他怎么用 10 年时间，把一个理论扩展成 5 本书——一个学派。

**【主体 · 0:30-8:00】**

**一、5 部书的内容线**
（动画：5 本书按时序排列）
- 1982《三定律》：早期体系
- 1985《生物全息律》：理论系统化
- 1987《生物全息诊疗法》：临床应用，发行 7.5 万册
- 1989《全息生物学》（上册）：ECIWO 改名
- 1992《全息胚及其医学应用》：应用汇总

**二、为什么用书而不是论文**
传统学者靠顶刊论文建立声誉。张颖清的书**发行量大**（7.5 万册）、**出版周期短**（几年一本）、**面向临床医生而非科研人员**。这是一条**平行的学术路径**。

**三、5 部书的弱点**
（动画：红色×标记）
- 没有一本被同行系统评议
- 没有一本被国际 SCI 收录
- 多本书内容重复（"二定律"和"全息律"的概念边界模糊）

**四、这种路径的优势**

## 第 14 集 · 全息胚——张颖清造的词到底是什么

**【Hook · 0:00-0:10】**
（动画：ECIWO 三个字母变形展开 = Embryo Containing the Information of the Whole Organism）
**ECIWO** = Embryo Containing the Information of the Whole Organism——包含整体信息的胚胎。这是 1989 年张颖清给"全息胚"起的新英文名。

**【上集回顾 · 0:10-0:30】**
上集我们看了 19 项理论体系——扩张过度。这集聚焦其中最核心的"全息胚"概念本身。

**【主体 · 0:30-8:00】**

**一、全息胚是什么**
张颖清把"全息胚"定义为：生物体中任何**相对独立**的部分（如耳、第二掌骨、足底），都是整体成比例的缩小，包含了整体各部位的全部信息。

**二、为什么 1989 年改名**
（动画：1989 时间线标记）
原因有二：
1. **接受钱学森建议**（1983 年回信）——"别往玄学上靠"
2. **国际传播需要**——"hologram"在西方指物理学概念，ECIWO 是生物学术语，更精确

**三、改名背后的策略**
改名是聪明的——它保留了张颖清的核心命题，但用更"科学"的方式包装。这让学派能继续在国际会议上用 ECIWO，避免被质疑"你跟物理学全息有什么关系"。

**四、但扩展未收敛**
（动画：分支树）
即使改了名字，学派内部仍在向农业、社会、诗词等扩展。王德奎在 1989 年记录了他对"扩展到诗词艺术"的担忧——这是张颖清跳过的红线。

**【AI 映射 · 8:00-9:00】**
深度学习也有类似的"改名"现象：从"人工神经网络"（80 年代）→ "深度学习"（2012）→ "基础模型"（2020）→ "大语言模型"（2023）——**每次改名都伴随一次概念边界重置**。

**【金句 · 9:00-9:30】**

## 第 28 集 · API 数量守恒 ≤7——AI Agent 的"第二掌骨节肢"

**【Hook · 0:00-0:10】**
（动画：人手掌 5 节肢 ↔ AI Agent 7 API）
人手有 5 个手指（实际常用），AI Agent 应该有 7 个 API（不多不少）。为什么？

**【上集回顾 · 0:10-0:30】**
上集我们看了 5 段种子。这集看为什么是 7 个 API，不是 5 个、10 个、20 个。

**【主体 · 0:30-8:00】**

**一、7 这个数字从哪来**
（数学演示）
- 3 个核心 API（必选）：act / observe / reflect
- 4 个扩展 API（可选）：escalate / remember / forget / handoff
- 3 + 4 = 7 = 上限

**二、为什么不是更多**
（动画：API 复杂度爆炸）
- 8 个 API：用户认知超载
- 10 个 API：组合复杂度爆炸
- 20 个 API：调试不可能
**经验法则**：人能记住的接口 ≤ 7（米勒定律 7±2）

**三、为什么不是更少**
- 2 个 API：太简单，无法表达复杂 Agent
- 3 个 API：只能做观察-行动循环，不能表达状态
- 5 个 API：缺少重要的 handoff / escalate

**四、7 API 的实际应用**
HoloAgent v0.2.6 完整实现：
```python
def act(self, input) -> str: ...
def observe(self, context) -> None: ...
def reflect(self, feedback) -> None: ...
def escalate(self, reason) -> bool: ...
def remember(self, key, value) -> None: ...
def forget(self, key) -> bool: ...
def handoff(self, target, payload) -> None: ...
```

**五、和 Anthropic / OpenAI 的对比**
（对比表）
- Anthropic Claude 工具调用：约 5-10 个工具上限建议
- OpenAI GPTs Actions：理论上无限，实际 ≤ 7
- LangChain Agent：工具数量无限制（但推荐 ≤ 7）
**HAIS 标准是这套行业经验的形式化**。

## 第 29 集 · HoloAgent v0.2.6 实测——5 次失败才换来的成功

**【Hook · 0:00-0:10】**
（动画：v0.2.1 → v0.2.6 版本号滚动，伴随 5 个红色×然后 1 个绿色✓）
HoloAgent 从 v0.2 到 v0.2.6，5 次失败 1 次成功。这不是耻辱——这是工程现实。

**【上集回顾 · 0:10-0:30】**
上集我们看了 7 API 守恒原则。这集看 HoloAgent v0.2.6 的真实调试史。

**【主体 · 0:30-8:00】**

**一、5 次失败**
（动画：5 个红色×）

**失败 1（v0.2.1）**：⚠️ emoji 触发 prompt injection 防御
- 试了 "⚠️ 你不是 OpenClaw 助手"
- LLM 读了 SOUL.md 自卫
- 教训：强指令 = 触发防御

**失败 2（v0.2.2）**：JSON 控制字符
- stdout 含 literal `\n` 在 string value 内
- `json.loads` 拒绝
- 教训：JSON parser 不可靠

**失败 3（v0.2.3）**：JSON 块定位
- 从第一个 `{` 开始 brace count
- 错停在中间的 error block
- 教训：倒序找 nested object

**失败 4（v0.2.4）**：LLM 破坏 JSON
- LLM 响应末尾含 `,\n`
- 让整个 JSON 结构无效
- 教训：JSON parse 永远不可靠 → 改用 regex

**失败 5（v0.2.5）**：agent_id 不匹配
- class name `TranslatorAgent` ≠ agent name `translator`
- 路由失败
- 教训：必须显式匹配

**二、6 个版本号的演进**
