## 1. 目标用户（3 层）

### 1.1 主要用户（健康焦虑的中产）
- **画像**: 30-50 岁一二线城市白领，年收入 20-50 万
- **痛点**: 工作忙没时间去医院，但身体偶有不适（失眠、消化差、颈椎痛）
- **现有方案**: 体检（一年一次）、保健品（事后补救）、百度自查（越查越慌）
- **HoloScan 提供**: 每天 30 秒"自检"，早期预警，不焦虑
- **市场规模**: 中国 30-50 岁中产约 **1.5 亿人**，按 5% 付费意愿 = **750 万付费用户**

### 1.2 次要用户（中医诊所 / 推拿店）
- **画像**: 中小型中医诊所、推拿店、养生馆
- **痛点**: 缺乏标准化诊断工具，老中医经验难以传承，年轻医生经验不足
- **现有方案**: 望闻问切（主观、依赖经验）、西医仪器（贵、不符合中医逻辑）
- **HoloScan 提供**: 标准化"全息诊断"辅助工具，年轻医生也能用
- **市场规模**: 中国中医诊所约 **8 万家**，按 10% 渗透 = **8000 家 B 端客户**

### 1.3 长期用户（健康险公司 / 体检中心）
- **画像**: 保险公司、健康管理公司、连锁体检中心
- **痛点**: 客户日常健康数据缺乏，难以做风险评估
- **HoloScan 提供**: 用户授权后的连续健康数据流
- **市场规模**: 中国健康险公司约 **150 家**，每家可采购数千万级

## 2. 产品形态（3 形态组合）

### 形态 A: HoloScan App（消费级 C 端）
- **平台**: iOS / Android
- **核心功能**:
  - 第二掌骨扫描：摄像头拍掌骨侧位，AI 识别掌骨节肢系统的穴位状态（颜色、丘疹、压痛点）
  - 耳廓扫描：摄像头拍耳朵，AI 识别耳穴反应点
  - 30 秒生成"全息健康报告"：可能存在问题的器官/系统 + 建议（就医/调理/观察）
  - 历史对比：每日扫描数据对比，看趋势变化
- **价格**:
  - 免费版：每日 1 次扫描，基础报告
  - 会员版：¥29/月，无限扫描 + 详细报告 + 中医调理建议
  - 家庭版：¥99/月，5 个账号共享

### 形态 B: HoloScan Pro（诊所级 B 端）
- **形态**: Pad 应用 + 配套高清摄像头 + 报告打印机
- **核心功能**:
  - 比 App 更精准的图像识别（医疗级摄像头）
  - 完整的"全息诊断"工作流（病史记录、复诊对比、报告导出）
  - 中医师培训课程（线上 + 线下）
  - 客户管理（CRM）+ 复购提醒
- **价格**:
  - 一次性硬件：¥3,000-8,000（Pad + 摄像头 + 打印机）
  - SaaS 年费：¥3,600-12,000/年（按诊所规模）
  - 培训费：¥2,000/人（2 天线下）

### 形态 C: HoloScan Health Data（数据服务）
- **形态**: API + 数据看板
- **核心功能**:
  - 为保险公司/体检中心提供"连续健康数据流"API
  - 群体健康趋势分析（脱敏）
  - 异常事件预警
- **价格**: 按调用次数 ¥0.1-1/次，年度合同百万级

## 3. 技术方案（关键路径）

### 3.1 核心 AI 模型
- **图像识别**: 第二掌骨 / 耳廓关键点检测（基于 ResNet / ViT）
- **状态识别**: 穴位颜色、丘疹、压痛点的视觉特征分类
- **推理引擎**: 把视觉特征映射到"对应器官/系统"
  - **核心知识库**: laimengjun学派的"元DNA"映射表（基于张秀勤《全息经络刮痧法》整理）
  - **现代医学映射**: 与 ICD-10 / 中医证候分类对应
- **LLM 集成**: 用 HoloAgent (HAIS) 架构，每个用户有一个"健康 Agent"

### 3.2 数据采集（**冷启动关键**）
- **第一阶段（2026-2027）**: 与 3-5 家中医诊所合作，采集 5,000+ 标注样本
- **第二阶段（2027-2028）**: 上线 App，让用户自标注 + 中医师审核
- **第三阶段（2028+）**: 达到 100 万级样本后，AI 准确率应该接近资深中医师

### 3.3 关键风险点
- **诊断准确性**: 全息诊法本身是经验医学，AI 能否超越资深中医？**未知**
- **监管风险**: 中国对"医疗诊断"类 App 有严格资质要求（医疗器械注册）
- **学术争议**: laimengjun学派本身就是边缘学科，商业化会被同行质疑

## 9. 与 HoloAgent (HAIS) 的整合

这是 HoloScan 的**独特技术差异化**：

```
HoloScan 用户 App
    ↓ 每个用户
HoloAgent (HAIS 自包含)
    ├── 5 段种子：身份/能力/边界/语气/升级
    ├── 3 个核心 API: act/observe/reflect
    └── 7 个扩展 API: escalate/remember/forget/handoff
        ↓ 协作
HoloScan 健康分析引擎 (中心 LLM)
    ├── 第二掌骨识别
    ├── 耳穴识别
    └── 全息诊断知识库
```

**关键优势**:
- 每个用户的健康数据**完全隔离**（Agent 私有记忆）
- Agent 可独立调用健康分析引擎（不暴露用户数据）
- 多用户协作场景：家庭医生 Agent + 个人 Agent 协同
