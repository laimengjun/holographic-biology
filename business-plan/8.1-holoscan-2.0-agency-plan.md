**HoloScan 2.0 = 张颖清全息胚理论的工程化 AI 平台**

**四大核心目的**：
1. **临床价值**：把张颖清全息胚理论工程化、产品化、AI 化
2. **杀手级应用**：6-24 个月癌前病变预警（检出率 30-40% → 70-80%）
3. **商业价值**：TAM ~¥3000 亿/年
4. **科学贡献**：建立"全息胚疗法家族"的统一工程平台

## 五、agent 协作工作流

### 5.1 单 Agent 工作流（日常任务）

```
用户任务 → orchestrator 路由 → 单一 Agent 执行 → 交付
```

### 5.2 多 Agent 编排工作流（复杂项目）

```
senior-pm 任务分解
   ↓
backend-architect 架构设计
   ↓
并行：
├─ mobile-app-builder 实现移动端
├─ frontend-developer 实现 Web
├─ ai-engineer 实现 AI
   ↓
evidence-collector 视觉 QA
api-tester 集成测试
   ↓
reality-checker 最终 QA
   ↓
analytics-reporter 数据分析
executive-summary-generator 高管摘要
   ↓
交付
```

### 5.3 失败处理工作流

```
reality-checker 发现缺陷
   ↓
senior-pm 重新分解任务
   ↓
原 Agent 重试（最多 3 次）
   ↓
若仍失败 → 切换备选 Agent
   ↓
orchestrator 记录失败 → 风险预警
```
