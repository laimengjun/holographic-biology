# 8.8 HAIS v0.2 设计文档

> **版本**: v0.2-draft
> **日期**: 2026-07-03
> **作者**: AI 起草
> **状态**: 设计阶段
> **目标**: 解决 v0.1 的核心局限（记忆不隔离 + 无跨 Agent 消息）
> **核心命题**: 5 段种子 + 7 API 升级为"记忆隔离 + 消息总线 + 多 Agent 协作"

## 一、v0.1 现状回顾

### 1.1 v0.1 已实现（2026-07-01）

✅ 5 段知识种子：
- identity（身份）
- capability（能力）
- boundary（边界）
- voice（语音）
- escalation（升级）

✅ 3 必选 + 4 扩展 API（共 7）：
- act() / observe() / reflect()（必选）
- escalate() / remember() / forget() / handoff()（扩展）

✅ API 数量守恒 ≤7

✅ 单文件部署优先

✅ HoloAgent v0.2.6 实战验证
- TranslatorAgent / ProofreaderAgent 角色代入完美
- latency 30-40s

### 1.2 v0.1 的核心局限

**局限 1：记忆不隔离**
- 所有 Agent 共享 OpenClaw main session 的上下文
- Agent A 看到 Agent B 的对话历史
- 隐私 + 安全风险

**局限 2：无跨 Agent 消息总线**
- Agent 之间只能通过 handoff() 传递
- 不能实时协作
- 不能共享数据

**局限 3：handoff() 不支持多对多**
- v0.1 handoff() 是一对一
- 不能 1-to-N 广播
- 不能 N-to-1 汇聚

**局限 4：无持久化记忆**
- 每次 Agent 启动是"全新"的
- 没有"经验积累"
- 不能学习

**局限 5：无标准化的 Agent 身份认证**
- Agent 之间的"信任"机制缺失
- 容易出现 Agent 之间的混乱

#### 3.3.1 消息格式


|------|------|------|

## 四、API 升级

### 4.1 7 API 保留 + 1 新增（v0.2 守恒边界）

**v0.1 保留**：
- act() / observe() / reflect()（必选）
- escalate() / remember() / forget() / handoff()（扩展）

**v0.2 新增 1 个核心 API**：
- **collaborate()** —— 跨 Agent 协作（取代或扩展 handoff()）

> **API 守恒突破**：v0.2 引入**内部 API**（agent 内部）和**外部 API**（跨 agent）：
> - 内部 API：act / observe / reflect / escalate / remember / forget（6 个）
> - 外部 API：collaborate（1 个）
> - 守恒总数 = 7

### 4.2 collaborate() 设计

```python
# 广播给多个 Agent
await collaborate.broadcast(
    topic="task:new",
    payload={"input": "..."}
)

# 等待所有 Agent 响应
responses = await collaborate.gather(
    agent_ids=["proofreader", "summarizer"],
    timeout=30
)

# 顺序处理
result = await collaborate.chain(
    agents=["translator", "proofreader", "summarizer"],
    payload={"input": "..."}
)
```

### 4.3 协作模式

| 模式 | 描述 | 适用 |
|------|------|------|
| **顺序** | A → B → C | 流水线任务 |
| **并行** | A + B + C → 汇聚 | 独立子任务 |
| **投票** | A + B + C → 多数决 | 决策 |
| **共识** | A + B + C → 全员同意 | 关键决策 |
| **层次** | orchestrator → sub-agents | 复杂任务 |

## 五、orchestrator 设计

### 5.1 核心职责

```yaml
orchestrator:
  role: 任务分解 + Agent 调度 + 依赖管理
  capabilities:
    - task_decomposition: 把复杂任务分解为子任务
    - agent_selection: 根据子任务选择最合适的 Agent
    - dependency_management: 管理子任务的依赖关系
    - result_aggregation: 汇聚子任务的结果
    - error_recovery: 处理 Agent 失败
    - load_balancing: 平衡 Agent 负载
```

### 5.2 工作流示例

```yaml
task: "翻译 100 页中文技术文档到英文"

decomposition:
  - subtask: "翻译"
    agent: translator
    input: "中文技术文档"
    output: "英文翻译稿"
  - subtask: "校对"
    agent: proofreader
    input: "英文翻译稿"
    output: "校对后英文稿"
  - subtask: "摘要"
    agent: summarizer
    input: "校对后英文稿"
    output: "英文摘要"

dependencies:
  proofreader: depends_on: [translator]
  summarizer: depends_on: [proofreader]
```

### 5.3 orchestrator 实现

```python
class Orchestrator:
    def __init__(self, agent_registry):
        self.registry = agent_registry
        self.bus = HoloMessageBus()
    
    def decompose(self, task: Task) -> List[SubTask]:
        """LLM-based 任务分解"""
        return self.llm.decompose(task)
    
    def select_agents(self, subtasks: List[SubTask]) -> List[Agent]:
        """选择每个子任务最合适的 Agent"""
        return [self.registry.find_best(s) for s in subtasks]
    
    async def execute(self, plan: Plan) -> Result:
        """按依赖关系执行"""
        for stage in plan.stages:
            # 同一 stage 并行
            results = await asyncio.gather(*[
                self.execute_subtask(s) for s in stage.subtasks
            ])
            # 检查失败
            if any(r.failed for r in results):
                return self.handle_failure(results)
        return self.aggregate(plan)
```

### 6.3 长期记忆 Schema

```sql

  id SERIAL PRIMARY KEY,
  agent_id TEXT NOT NULL,
  embedding VECTOR(1536),    -- OpenAI / minimax embedding
  content TEXT NOT NULL,

  created_at TIMESTAMP DEFAULT NOW(),
  last_accessed_at TIMESTAMP,
  access_count INT DEFAULT 0
```

### 6.4 记忆生命周期

```
工作记忆（7±2 项）
  ↓ 超过容量
短期记忆（SQLite，按 session 隔离）
  ↓ 重要性 > 阈值
长期记忆（PostgreSQL + pgvector）
  ↓ 1 年未访问
归档（冷存储）
```

### 6.5 重要性评分

```python
def calculate_importance(message):
    score = 0.0
    # 重复出现 +0.2
    if is_repeated(message): score += 0.2
    # 用户标记为重要 +0.5
    if message.metadata.get('user_important'): score += 0.5
    # 包含实体（人名/项目/数字）+0.2
    if has_entities(message): score += 0.2
    # Agent 反思后认为是关键 +0.3
    if is_reflection_key(message): score += 0.3
    return min(score, 1.0)
```

## 七、消息总线实现

### 7.1 协议栈

```
┌────────────────────┐
│  HoloMessage v1.0  │  ← 应用层（JSON 消息）
├────────────────────┤
│  WebSocket / HTTP   │  ← 传输层
├────────────────────┤
│  TCP / Unix Socket  │  ← 网络层
└────────────────────┘
```

### 7.2 消息总线架构

```python
class HoloMessageBus:
    def __init__(self, port=8765):
        self.port = port
        self.subscribers = {}  # topic → [agent_id]
        self.routes = {}       # agent_id → connection
        self.queue = asyncio.Queue()
    
    async def start(self):
        """启动消息总线"""
        self.server = await websockets.serve(self.handle, "0.0.0.0", self.port)
    
    async def handle(self, ws, path):
        """处理连接"""
        agent_id = await self.authenticate(ws)
        async for msg in ws:
            await self.route(agent_id, msg)
    
    async def route(self, from_agent, msg):
        """路由消息"""
        if msg.to == "*":
            # 广播
            for sub in self.subscribers.get(msg.topic, []):
                await self.send_to(sub, msg)
        else:
            # 直发
            await self.send_to(msg.to, msg)
```

### 7.3 Agent 客户端

```python
class HoloAgent:
    def __init__(self, agent_id, bus_url="ws://localhost:8765"):
        self.agent_id = agent_id
        self.bus = await websockets.connect(bus_url)
    
    async def collaborate(self, topic, payload):
        """发送协作消息"""
        msg = {
            "from": self.agent_id,
            "to": "*",
            "topic": topic,
            "payload": payload,
            "type": "request"
        }
        await self.bus.send(json.dumps(msg))
    
    async def listen(self, handler):
        """监听消息"""
        async for raw in self.bus:
            msg = json.loads(raw)
            await handler(msg)
```

### 8.2 消息签名

```python
import hmac
import hashlib

def sign(message, secret_key):
    msg_str = json.dumps(message, sort_keys=True)
    return hmac.new(
        secret_key.encode(),
        msg_str.encode(),
        hashlib.sha256
    ).hexdigest()

def verify(message, signature, secret_key):
    expected = sign(message, secret_key)
    return hmac.compare_digest(expected, signature)
```

### 8.3 加密（可选）

```python
from cryptography.fernet import Fernet

# 敏感消息加密
def encrypt_message(payload, key):
    f = Fernet(key)
    return f.encrypt(json.dumps(payload).encode())

def decrypt_message(encrypted, key):
    f = Fernet(key)
    return json.loads(f.decrypt(encrypted))
```
