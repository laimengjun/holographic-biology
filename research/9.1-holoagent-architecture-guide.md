## 1. HAIS v0.2 核心思想回顾

### 1.1 全息胚类比

> 参考：8.8-hais-v0.2-design.md §一

HAIS（Holographic Agent Interaction Standard）的核心类比来自全息生物学：**一个 Agent 就是一个数字全息胚**。

在生物体中，一个受精卵包含生物体的全部信息；在 HAIS 中，一个 Agent 的知识种子（5 段种子）定义了它的全部行为特征：

| 种子段 | 生物学类比 | Agent 映射 |
|--------|-----------|-----------|
| **identity** | DNA 身份标记 | Agent ID、显示名、创建时间 |
| **capability** | 基因表达图谱 | 它能做什么、专长领域 |
| **boundary** | 细胞膜通透性 | 什么不该做、硬件限制 |
| **voice** | 表型特征 | 语气、风格、沟通方式 |
| **escalation** | 免疫应答 | 什么情况下向谁求助 |

**7 个 API = 7 个穴位**：每个 API 是对外暴露的功能"穴道"，外部通过"穴位刺激"来调用 Agent 的能力。v0.2 的 API 守恒原则为 **6 个内部 API + 1 个外部 API = 7 个**（参见 §4.1 设计文档）。

### 1.2 v0.2 vs v0.1：关键升级

> 参考：8.8-hais-v0.2-design.md §十四

| 维度 | v0.1 | v0.2 |
|------|------|------|
| **记忆** | 共享上下文 | **3 层隔离记忆**（工作/短期/长期） |
| **跨 Agent 通信** | handoff() 一对一 | **消息总线 + collaborate() 多对多** |
| **协作模式** | 仅 1-to-1 接力 | **四种模式：链/并行/投票/共识** |
| **持久化** | 无（每次全新） | **跨 session 经验保留** |
| **身份认证** | 弱（无签名） | **强（HMAC 签名 + 可选加密）** |
| **API 守恒** | 7 内部 | **6 内部 + 1 外部 = 7** |

### 1.3 一句话总结

> **HAIS v0.2 = 记忆隔离 + 消息总线 + 四种协作模式 + 身份认证**

## 2. 记忆系统实现指南

### 2.1 三层记忆架构

> 参考：8.8-hais-v0.2-design.md §3.2, §6

HAIS v0.2 定义了三个记忆层级，按**速度 → 容量 → 持久性**排布：

```
工作记忆（7±2 项，纳秒级）  ← Redis / 内存字典
  ↓ 超过容量或 session 结束
短期记忆（1MB/Agent，毫秒级）  ← SQLite
  ↓ 重要性 > 阈值
长期记忆（无限，秒级）  ← PostgreSQL + pgvector
  ↓ 1 年未访问
归档（冷存储）
```

### 2.2 最小可行实现：纯 SQLite 方案

> 不想上 PostgreSQL 和 Redis？**纯 SQLite 也能跑**。

```python
"""
hais_memory.py — HAIS v0.2 最小记忆系统
无外部依赖，仅需 Python 3.8+ 内置 sqlite3
"""
import sqlite3
import json
import time
import hashlib
from typing import Optional, List, Dict
from dataclasses import dataclass, field

# ─── 数据结构 ─────────────────────────────────────────────

@dataclass
class MemoryEntry:
    agent_id: str
    session_id: str
    role: str          # user / assistant / system
    content: str
    metadata: dict = field(default_factory=dict)
    importance: float = 0.0
    created_at: float = 0.0

# ─── 三层记忆代理 ─────────────────────────────────────────

class WorkingMemory:
    """工作记忆：纯内存字典，无持久化"""
    MAX_ITEMS = 9  # 7±2
    
    def __init__(self):
        self._store: Dict[str, str] = {}
        self._order: List[str] = []
    
    def set(self, key: str, value: str):
        if key in self._store:
            self._order.remove(key)
        self._store[key] = value
        self._order.append(key)
        # 超出容量，移除最早
        if len(self._order) > self.MAX_ITEMS:
            old = self._order.pop(0)
            del self._store[old]
    
    def get(self, key: str) -> Optional[str]:
        return self._store.get(key)
    
    def clear(self):
        self._store.clear()
        self._order.clear()


class ShortTermMemory:
    """短期记忆：SQLite，按 agent_id+session_id 隔离"""
    
    def __init__(self, db_path: str = "hais_memory.db"):
        self.conn = sqlite3.connect(db_path)
        self._init_db()
    
    def _init_db(self):
        self.conn.execute("""
            CREATE TABLE IF NOT EXISTS short_term_memory (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                agent_id TEXT NOT NULL,
                session_id TEXT NOT NULL,
                role TEXT NOT NULL,
                content TEXT NOT NULL,
                metadata TEXT DEFAULT '{}',
                importance REAL DEFAULT 0.0,
                created_at REAL DEFAULT (strftime('%s','now'))
            )
        """)
        self.conn.execute("""
            CREATE INDEX IF NOT EXISTS idx_stm_agent_session
            ON short_term_memory(agent_id, session_id)
        """)
        self.conn.commit()
    
    def append(self, agent_id: str, session_id: str,
               role: str, content: str,
               metadata: Optional[dict] = None):
        """添加一条短期记忆"""
        self.conn.execute(
            "INSERT INTO short_term_memory "
            "(agent_id, session_id, role, content, metadata, importance) "
            "VALUES (?, ?, ?, ?, ?, ?)",
            (agent_id, session_id, role, content,
             json.dumps(metadata or {}),
             _compute_importance(content, metadata or {}))
        )
        self.conn.commit()
    
    def search(self, agent_id: str, session_id: str,
               query: str = "", top_k: int = 10) -> List[Dict]:
        """关键词搜索短期记忆（无 embedding 的轻量方案）"""
        if query:
            rows = self.conn.execute(
                """SELECT role, content, metadata, importance, created_at
                   FROM short_term_memory
                   WHERE agent_id = ? AND session_id = ?
                     AND content LIKE ?
                   ORDER BY created_at DESC LIMIT ?""",
                (agent_id, session_id, f"%{query}%", top_k)
            ).fetchall()
        else:
            rows = self.conn.execute(
                """SELECT role, content, metadata, importance, created_at
                   FROM short_term_memory
                   WHERE agent_id = ? AND session_id = ?
                   ORDER BY created_at DESC LIMIT ?""",
                (agent_id, session_id, top_k)
            ).fetchall()
        return [
            {"role": r[0], "content": r[1],
             "metadata": json.loads(r[2]),
             "importance": r[3], "created_at": r[4]}
            for r in rows
        ]
    
    def close(self):
        self.conn.close()


class LongTermMemory:
    """长期记忆：SQLite 基础版（无向量搜索，用关键词替代）
    生产环境请迁移到 PostgreSQL + pgvector"""
    
    def __init__(self, db_path: str = "hais_memory.db"):
        self.conn = sqlite3.connect(db_path)
        self._init_db()
    
    def _init_db(self):
        self.conn.execute("""
            CREATE TABLE IF NOT EXISTS long_term_memory (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                agent_id TEXT NOT NULL,
                content TEXT NOT NULL,
                metadata TEXT DEFAULT '{}',
                importance REAL DEFAULT 0.5,
                created_at REAL DEFAULT (strftime('%s','now')),
                last_accessed_at REAL,
                access_count INTEGER DEFAULT 0
            )
        """)
        self.conn.execute("""
            CREATE INDEX IF NOT EXISTS idx_ltm_agent
            ON long_term_memory(agent_id)
        """)
        self.conn.commit()
    
    def store(self, agent_id: str, content: str,
              metadata: Optional[dict] = None,
              importance: float = 0.5):
        """存入长期记忆"""
        self.conn.execute(
            "INSERT INTO long_term_memory "
            "(agent_id, content, metadata, importance) "
            "VALUES (?, ?, ?, ?)",
            (agent_id, content, json.dumps(metadata or {}), importance)
        )
        self.conn.commit()
    
    def recall(self, agent_id: str, query: str = "",
               top_k: int = 5) -> List[Dict]:
        """召回长期记忆（关键词匹配，生产环境请用向量搜索）"""
        if query:
            rows = self.conn.execute(
                """SELECT content, metadata, importance, access_count, created_at
                   FROM long_term_memory
                   WHERE agent_id = ? AND content LIKE ?
                   ORDER BY importance DESC, access_count DESC
                   LIMIT ?""",
                (agent_id, f"%{query}%", top_k)
            ).fetchall()
        else:
            rows = self.conn.execute(
                """SELECT content, metadata, importance, access_count, created_at
                   FROM long_term_memory
                   WHERE agent_id = ?
                   ORDER BY importance DESC, access_count DESC
                   LIMIT ?""",
                (agent_id, top_k)
            ).fetchall()
        # 更新访问计数
        for r in rows:
            self.conn.execute(
                "UPDATE long_term_memory SET access_count = access_count + 1, "
                "last_accessed_at = strftime('%s','now') "
                "WHERE content = ? AND agent_id = ?",
                (r[0], agent_id)
            )
        self.conn.commit()
        return [
            {"content": r[0], "metadata": json.loads(r[1]),
             "importance": r[2], "access_count": r[3], "created_at": r[4]}
            for r in rows
        ]
    
    def close(self):
        self.conn.close()


class HaisMemory:
    """统一记忆 API：封装三层记忆"""
    
    def __init__(self, agent_id: str, session_id: str,
                 db_path: str = "hais_memory.db"):
        self.agent_id = agent_id
        self.session_id = session_id
        self.working = WorkingMemory()
        self.short = ShortTermMemory(db_path)
        self.long = LongTermMemory(db_path)
    
    def append(self, role: str, content: str,
               metadata: Optional[dict] = None):
        """添加一条记忆（自动分配到三层）"""
        # 1. 工作记忆：保留最近上下文
        self.working.set(f"msg:{len(self.working._order)}", content)
        
        # 2. 短期记忆：持久化存储
        self.short.append(
            self.agent_id, self.session_id, role, content, metadata
        )
        
        # 3. 长期记忆：重要性 > 0.7 时进入
        imp = _compute_importance(content, metadata or {})
        if imp >= 0.7:
            self.long.store(self.agent_id, content, metadata, imp)
    
    def close(self):
        self.short.close()
        self.long.close()


# ─── 重要性评分 ───────────────────────────────────────────

def _compute_importance(content: str,
                        metadata: dict) -> float:
    """
    记忆重要性评分公式
    
    规则（参考 8.8-hais-v0.2-design.md §6.5）：
    - 基础分 0.1
    - 用户标记重要 +0.5
    - 包含数字/金额/人名 +0.2
    - 用户反思或总结 +0.3
    - 含项目/概念关键词 +0.15
    - 上限 1.0
    """
    score = 0.1  # 基础分
    
    if metadata.get("user_important"):
        score += 0.5
    
    # 含实体（数字、关键词）
    import re
    if re.search(r'\d+', content):
        score += 0.15
    for kw in ["总结", "关键", "重要", "记住", "核心",
               "decision", "key", "important", "记住"]:
        if kw in content.lower():
            score += 0.15
            break
    if metadata.get("role") in ("assistant",) and len(content) > 200:
        score += 0.2  # 长回复可能包含有价值信息
    
    return min(score, 1.0)


# ─── 快速测试 ─────────────────────────────────────────────

if __name__ == "__main__":
    # 使用示例
    mem = HaisMemory("translator", "session:abc")
    
    # 写入
    mem.append("user", "请翻译这段中医文献：脉象浮紧，舌苔白滑",
               {"user_important": True})
    mem.append("assistant", "翻译结果：Pulse is floating and tight, "
               "tongue coating is white and slippery.",
               {"role": "assistant"})
    
    # 搜索短期记忆
    results = mem.short.search("translator", "session:abc", "舌苔")
    print(f"短期记忆命中：{len(results)} 条")
    for r in results:
        print(f"  [{r['role']}] {r['content'][:50]}...")
    
    # 搜索长期记忆
    long_results = mem.long.recall("translator")
    print(f"长期记忆命中：{len(long_results)} 条")
    
    mem.close()
```

### 2.3 记忆隔离的实现模式

> 参考：8.8-hais-v0.2-design.md §3.2.3

记忆隔离的核心是 **namespace**：

```python
# 每个 Agent 有独立的 namespace
namespace = f"agent:{agent_id}"

# 共享记忆需要显式授权
shared_ns = "global"
```

**SQLite 中隔离的两种方式**：

| 方式 | 做法 | 适用场景 |
|------|------|---------|
| **同一 DB，agent_id 列** | 所有 Agent 共享一个 .db 文件，SQL WHERE agent_id=? 隔离 | 轻量、单机 |
| **每个 Agent 独立 DB** | 每个 Agent 一个 .db 文件 | 高安全、多租户 |

推荐默认用**方式 1**（单文件）——对上文 `hais_memory.py` 来说，就是 `WHERE agent_id = ?` 这个条件。当你看到记忆泄漏问题时，再加一层 `WHERE session_id = ?`。

### 2.4 生产级升级：嵌入向量

短期用关键词搜索，长期必须上向量搜索。最简单的升级路线：

```python
# 从 SQLite 关键词 → 本地向量搜索（无 GPU 也能跑）
# pip install sentence-transformers chromadb

from sentence_transformers import SentenceTransformer

class VectorMemory:
    def __init__(self):
        self.encoder = SentenceTransformer(
            "paraphrase-multilingual-MiniLM-L12-v2"  # 支持中文
        )
    
    def embed(self, text: str) -> list:
        return self.encoder.encode(text).tolist()
    
    def search(self, query: str, embeddings: list,
               top_k: int = 5) -> list:
        """余弦相似度搜索"""
        q_vec = self.embed(query)
        scores = [
            (cosine_similarity(q_vec, e), idx)
            for idx, e in enumerate(embeddings)
        ]
        scores.sort(reverse=True)
        return scores[:top_k]

import math
def cosine_similarity(a, b):
    dot = sum(x*y for x, y in zip(a, b))
    norm_a = math.sqrt(sum(x*x for x in a))
    norm_b = math.sqrt(sum(x*x for x in b))
    return dot / (norm_a * norm_b + 1e-8)
```

**何时升级到 pgvector**？当你的记忆条目超过 10,000 条、搜索延迟 >200ms 时，把 SQLite 换成 PostgreSQL + pgvector。在此之前，关键词搜索 + 本地 embedding 足以胜任。

## 3. 消息总线实现模式

### 3.1 HoloMessage Protocol v1.0

> 参考：8.8-hais-v0.2-design.md §3.3, §7

HAIS 的所有跨 Agent 通信一律走 **HoloMessage** 格式：

```json
{
  "id": "msg-1721385910-a1b2c3",
  "version": "1.0",
  "from": {
    "agent_id": "translator",
    "session_id": "agent:translator:sess_01"
  },
  "to": {
    "agent_id": "proofreader",
    "session_id": "*"
  },
  "topic": "task:translate-and-proofread",
  "type": "request",
  "payload": {
    "input": "脉象浮紧，舌苔白滑",
    "language_from": "zh",
    "language_to": "en",
    "mode": "medical"
  },
  "metadata": {
    "priority": "normal",
    "ttl": 300,
    "trace_id": "trace-1721385910-xyz",
    "timestamp": 1721385910
  },
  "signature": "f8a7b6c5d4e3..."
}
```

**消息类型表**：

| 类型 | 用途 | 路由方式 |
|------|------|---------|
| `request` | 请求协作 | 直发 / 广播 |
| `response` | 回复请求 | 回给 from |
| `event` | 状态变更通知 | 广播 |
| `broadcast` | 通用广播 | 所有订阅者 |
| `query` | 查询 Agent 能力 | 广播 |
| `ack` | 确认收到 | 回给 from |
| `error` | 错误报告 | 回给 from |

### 3.2 最小可行实现：HTTP 消息总线

> 不想上 WebSocket？**HTTP 轮询 + 内存队列**也能跑。

```python
"""
hais_bus.py — HAIS v0.2 最小消息总线
无外部依赖，仅 Python 3.8+ 内置 http.server
"""
import json
import uuid
import time
import hmac
import hashlib
from http.server import HTTPServer, BaseHTTPRequestHandler
from urllib.parse import urlparse, parse_qs
from typing import Dict, List, Optional
from collections import defaultdict
import threading

# ─── HoloMessage 核心 ─────────────────────────────────────

def make_message(from_agent: str, to_agent: str,
                 topic: str, msg_type: str,
                 payload: dict,
                 secret_key: str = "") -> dict:
    """创建一条 HoloMessage"""
    msg = {
        "id": f"msg-{int(time.time())}-{uuid.uuid4().hex[:8]}",
        "version": "1.0",
        "from": {
            "agent_id": from_agent,
            "session_id": f"agent:{from_agent}:{uuid.uuid4().hex[:6]}"
        },
        "to": {
            "agent_id": to_agent,
            "session_id": "*"
        },
        "topic": topic,
        "type": msg_type,
        "payload": payload,
        "metadata": {
            "priority": "normal",
            "ttl": 300,
            "trace_id": f"trace-{int(time.time())}-{uuid.uuid4().hex[:8]}",
            "timestamp": int(time.time())
        },
        "signature": ""
    }
    if secret_key:
        msg["signature"] = _sign(msg, secret_key)
    return msg


def _sign(message: dict, secret_key: str) -> str:
    """HMAC-SHA256 签名（参考 8.8-hais-v0.2-design.md §8.2）"""
    payload = json.dumps(message, sort_keys=True, ensure_ascii=False)
    return hmac.new(
        secret_key.encode(),
        payload.encode(),
        hashlib.sha256
    ).hexdigest()


def verify(message: dict, secret_key: str) -> bool:
    """验证签名"""
    sig = message.pop("signature", "")
    expected = _sign(message, secret_key)
    message["signature"] = sig
    return hmac.compare_digest(expected, sig)


# ─── 消息总线服务器 ───────────────────────────────────────

class HoloMessageBus:
    """
    轻量 HTTP 消息总线
    
    路由规则（参考 8.8-hais-v0.2-design.md §3.3.3）：
    - 直发：to.agent_id 为具体 Agent ID
    - 广播：to.agent_id 为 "*"
    - Topic 订阅：按 topic 分发
    """
    
    def __init__(self, host="0.0.0.0", port=8765,
                 secret_key: str = "hais-dev-key"):
        self.host = host
        self.port = port
        self.secret_key = secret_key
        # queue[agent_id] = [messages]
        self.queues: Dict[str, List[dict]] = defaultdict(list)
        # subscriptions[topic] = {agent_id}
        self.subscriptions: Dict[str, set] = defaultdict(set)
        self._lock = threading.Lock()
    
    def _route(self, msg: dict):
        """路由一条消息"""
        to = msg["to"]["agent_id"]
        with self._lock:
            if to == "*":
                # 广播：投递到该 topic 的所有订阅者
                topic = msg.get("topic", "")
                for agent_id in self.subscriptions.get(topic, set()):
                    if agent_id != msg["from"]["agent_id"]:  # 不发给自己
                        self.queues[agent_id].append(msg)
            else:
                # 直发
                self.queues[to].append(msg)
    
    def subscribe(self, agent_id: str, topic: str):
        """Agent 订阅 topic"""
        with self._lock:
            self.subscriptions[topic].add(agent_id)
    
    def unsubscribe(self, agent_id: str, topic: str):
        with self._lock:
            self.subscriptions[topic].discard(agent_id)
    
    def start(self):
        """启动 HTTP 服务器"""
        bus = self
        
        class Handler(BaseHTTPRequestHandler):
            def log_message(self, format, *args):
                pass  # 静默日志
            
            def _send_json(self, data, status=200):
                self.send_response(status)
                self.send_header("Content-Type", "application/json")
                self.send_header("Access-Control-Allow-Origin", "*")
                self.end_headers()
                self.wfile.write(json.dumps(data).encode())
            
            def do_POST(self):
                parsed = urlparse(self.path)
                length = int(self.headers.get("Content-Length", 0))
                body = json.loads(self.rfile.read(length))
                
                if parsed.path == "/send":
                    # 发送消息
                    msg = body
                    if bus.secret_key:
                        if not verify(msg, bus.secret_key):
                            self._send_json({"error": "invalid signature"}, 401)
                            return
                    bus._route(msg)
                    self._send_json({"status": "ok", "msg_id": msg["id"]})
                
                elif parsed.path == "/subscribe":
                    bus.subscribe(body["agent_id"], body["topic"])
                    self._send_json({"status": "subscribed"})
                
                elif parsed.path == "/poll":
                    # 轮询消息
                    agent_id = body["agent_id"]
                    with bus._lock:
                        msgs = list(bus.queues.get(agent_id, []))
                        bus.queues[agent_id] = []
                    self._send_json({"messages": msgs})
                
                else:
                    self._send_json({"error": "not found"}, 404)
            
            def do_OPTIONS(self):
                self.send_response(200)
                self.send_header("Access-Control-Allow-Origin", "*")
                self.send_header("Access-Control-Allow-Methods", "POST, OPTIONS")
                self.send_header("Access-Control-Allow-Headers", "Content-Type")
                self.end_headers()
        
        self.server = HTTPServer((self.host, self.port), Handler)
        print(f"[HAIS Bus] 启动于 http://{self.host}:{self.port}")
        self.server.serve_forever()


# ─── Agent 客户端 ─────────────────────────────────────────

class HoloAgent:
    """HAIS 兼容的 Agent 基类"""
    
    def __init__(self, agent_id: str,
                 bus_url: str = "http://localhost:8765",
                 secret_key: str = "hais-dev-key"):
        self.agent_id = agent_id
        self.bus_url = bus_url
        self.secret_key = secret_key
        self._memory = None
        self._message_handlers = {}
    
    def send(self, to_agent: str, topic: str,
             msg_type: str, payload: dict) -> dict:
        """发送一条消息"""
        import urllib.request
        msg = make_message(
            self.agent_id, to_agent,
            topic, msg_type, payload, self.secret_key
        )
        req = urllib.request.Request(
            f"{self.bus_url}/send",
            data=json.dumps(msg).encode(),
            headers={"Content-Type": "application/json"}
        )
        resp = json.loads(urllib.request.urlopen(req).read())
        return resp
    
    def poll(self) -> List[dict]:
        """轮询消息"""
        import urllib.request
        req = urllib.request.Request(
            f"{self.bus_url}/poll",

        resp = json.loads(urllib.request.urlopen(req).read())
        return resp.get("messages", [])
```

### 3.4 安全要点

> 参考：8.8-hais-v0.2-design.md §8

1. **永远签名**：即使在内网环境，也要签名。内网不是安全假设。
2. **拒绝未签名消息**：bus 端配置 `require_signature=True`，收到无签名消息直接 401。
3. **密钥轮换**：长期密钥（`sk-*`）用于 Agent 注册，session 密钥用于每次连接。每 24h 轮换一次 session 密钥。
4. **可选加密**：敏感 payload 用 Fernet 加密（参考 §8.3 设计文档）。
5. **限流**：每个 Agent 默认 100 msg/min，超过自动丢包。

## 4. 多 Agent 协作模式

### 4.1 collaborate() API

> 参考：8.8-hais-v0.2-design.md §4.2

HAIS v0.2 新增的 **collaborate()** 是外部 API（唯一对外暴露的跨 Agent 接口）。与 handoff() 的最大区别：**从 1 对 1 变成 N 对 N**。

### 4.2 四种协作模式

#### 模式 1：顺序链（Chain）

```
输入 → [Translator] → [Proofreader] → [Summarizer] → 输出
```

```python
async def chain(self, agents: list, payload: dict) -> dict:
    """顺序链：A → B → C"""
    result = payload
    for agent_id in agents:
        # 向当前 Agent 发送请求
        self.send(agent_id, "task:execute", "request", result)
        # 等待响应
        response = await self.wait_response(agent_id)
        result = response["payload"]
    return result
```

**适用场景**：翻译→校对→摘要、数据清洗管线、审批流程。

#### 模式 2：并行汇聚（Parallel Gather）

```
       ┌→ [Translator_ZH]
输入 ──┼→ [Translator_JP] ──→ 汇聚 → 输出
       └→ [Translator_KR]
```

```python
async def parallel_gather(self, agents: list,
                          payload: dict,
                          timeout: int = 30) -> dict:
    """并行汇聚：独立 Agent 同时执行，结果汇总"""
    tasks = []
    for agent_id in agents:
        self.send(agent_id, "task:execute", "request", payload)
        tasks.append(self.wait_response(agent_id, timeout))
    
    responses = await asyncio.gather(*tasks, return_exceptions=True)
    
    # 汇聚结果
    aggregated = {}
    for agent_id, resp in zip(agents, responses):
        if isinstance(resp, Exception):
            aggregated[agent_id] = {"error": str(resp)}
        else:
            aggregated[agent_id] = resp["payload"]
    
    return {"result": aggregated, "status": "complete"}
```

**适用场景**：多语言翻译、多传感器读数、并行搜索。

#### 模式 3：投票（Vote）

```
输入 → [Voter_A]
        [Voter_B] → 多数决 → 输出
        [Voter_C]
```

```python
def vote(self, agents: list, payload: dict,
         min_agreement: float = 0.6) -> dict:
    """投票：多数决"""
    votes = {}
    for agent_id in agents:
        self.send(agent_id, "task:vote", "request", payload)
    
    results = self.poll_all(agents)
    
    # 统计投票
    for agent_id, resp in results.items():
        answer = resp["payload"].get("choice", "")
        votes[answer] = votes.get(answer, 0) + 1
    
    # 多数决
    max_votes = max(votes.values())
    if max_votes >= min_agreement * len(agents):
        winner = max(votes, key=votes.get)
        return {"winner": winner, "votes": votes, "status": "decided"}
    else:
        return {"divergence": votes, "status": "undecided"}
```

**适用场景**：AI 诊断结果交叉验证、内容质量审核、代码审查。

#### 模式 4：共识（Consensus）

```
输入 → [Expert_A] ──┐
        [Expert_B] ──┤── 全员同意? → 输出
        [Expert_C] ──┘      ↓ 不同意
                        分歧 → 协调 → 重投
```

```python
def consensus(self, agents: list, payload: dict,
              max_rounds: int = 3,
              timeout: int = 60) -> dict:
    """共识：全员同意，有分歧就协调重投"""
    current_payload = payload
    
    for round_num in range(1, max_rounds + 1):
        results = self._collect_opinions(agents, current_payload)
        
        # 检查是否全员同意
        answers = [r["payload"].get("agreed", False) for _, r in results.items()]
        if all(answers):
            return {
                "status": "consensus",
                "round": round_num,
                "result": current_payload
            }
        
        # 分歧 → 展示分歧点，让协调 Agent 重新提案
        disagreements = [
            {"agent": a, "reason": r["payload"].get("reason", "")}
            for a, r in results.items()
            if not r["payload"].get("agreed", False)
        ]
        
        # 协调消息
        current_payload["disagreements"] = disagreements
        reconciliation = self._call_mediator(agents, current_payload)
        current_payload = reconciliation["payload"]
    
    return {"status": "deadlock", "rounds": max_rounds}
```

**适用场景**：关键诊断结论、安全审批、高风险决策（如药物推荐）。

### 4.3 Orchestrator 的轻量实现

> 参考：8.8-hais-v0.2-design.md §5

不需要 Docker Compose、不需要 Redis 队列。**一个 async 函数就能做 orchestrator**：

```python
"""
hais_orchestrator.py — 最小化 orchestrator
"""
import asyncio
from typing import Callable, Dict, List

class LightOrchestrator:
    """轻量级 orchestrator，不做持久化，不做复杂调度"""
    
    def __init__(self):
        self.agents: Dict[str, Callable] = {}
    
    def register(self, agent_id: str, handler: Callable):
        """注册 Agent 处理函数"""
        self.agents[agent_id] = handler
    
    def define_workflow(self, dag: dict) -> List[list]:
        """
        定义 DAG 工作流
        
        dag = {
            "translator": {"depends_on": []},
            "proofreader": {"depends_on": ["translator"]},
            "summarizer": {"depends_on": ["proofreader"]},
        }
        
        返回按 stage 分组的任务列表
        """
        stages = []
        remaining = set(dag.keys())
        completed = set()
        
        while remaining:
            stage = set()
            for agent_id in list(remaining):
                deps = set(dag[agent_id].get("depends_on", []))
                if deps.issubset(completed):
                    stage.add(agent_id)
            
            if not stage:
                raise ValueError(f"DAG 死锁：{remaining}")
            
            stages.append(list(stage))
            remaining -= stage
            completed |= stage
        
        return stages
    
    async def execute(self, dag: dict,
                      initial_payload: dict) -> Dict:
        """执行 DAG 工作流"""
        stages = self.define_workflow(dag)
        payload = initial_payload
        
        for stage in stages:
            if len(stage) == 1:
                # 顺序执行
                agent = stage[0]
                payload = await self.agents[agent](payload)
            else:
                # 同一 stage 并行
                tasks = [self.agents[a](payload) for a in stage]
                results = await asyncio.gather(*tasks)
                payload = dict(zip(stage, results))
        
        return {"result": payload, "stages": len(stages)}


# ─── 使用示例 ─────────────────────────────────────────────

async def translator(payload):
    """模拟翻译 Agent"""
    text = payload.get("input", "")
    return {"translated": f"[ZH]{text}"}

async def proofreader(payload):
    return {"proofread": payload["translated"] + " ✓"}

async def summarizer(payload):
    return {"summary": f"摘要: {payload['proofread'][:20]}..."}

async def demo():
    orch = LightOrchestrator()
    orch.register("translator", translator)
    orch.register("proofreader", proofreader)
    orch.register("summarizer", summarizer)
    
    dag = {
        "translator": {"depends_on": []},
        "proofreader": {"depends_on": ["translator"]},
        "summarizer": {"depends_on": ["proofreader"]},
    }
    
    result = await orch.execute(dag, {"input": "脉象浮紧"})
    print("工作流结果:", result)

# asyncio.run(demo())
```

### 4.4 协作模式速查表

| 要做什么 | 用哪个 | 实现复杂度 | 容错性 |
|---------|--------|-----------|--------|
| 流水线翻译+校对 | **顺序链** | ⭐ 低 | 高（断点重试） |
| 多语言翻译 | **并行汇聚** | ⭐⭐ 中 | 中（部分失败不影响） |
| 诊断结果交叉验证 | **投票** | ⭐ 低 | 高（多数通过） |
| 高风险安全审批 | **共识** | ⭐⭐⭐ 高 | 极高（一票否决） |

## 5. 从 v0.2.6 迁移到 HAIS v0.2 的路径

### 5.1 现有单文件 Agent 如何升级

当前 v0.2.6 的 HoloAgent 是一个 ~500 行的 Python 单文件。升级到 HAIS v0.2 不需要重写，只需**三层叠加**：

```
v0.2.6 Agent（单文件）          HAIS v0.2
┌──────────────────────┐    ┌──────────────────────┐
│  5 段种子 + 7 API    │ →  │  5 段种子 + 7 API    │  ← 保留
│  无记忆              │    │  + 三层记忆系统       │  ← 新增①
│  handoff() 一对一    │    │  + collaborate()      │  ← 新增②
│  无消息总线          │    │  + HoloMessage 总线   │  ← 新增③
└──────────────────────┘    └──────────────────────┘
```

**三步迁移法**：

**Step 1**：给现有 Agent 加上记忆系统（1 天）
```python
# 原 v0.2.6 Agent

    async def act(self, input_text):
        return llm_response(input_text)

# → 升级后
from hais_memory import HaisMemory

    def __init__(self, agent_id, session_id):
        self.agent_id = agent_id
        self.seeds = load_seeds()
        self.memory = HaisMemory(agent_id, session_id)  # 新增
    
    async def act(self, input_text):
        self.memory.append("user", input_text)
        result = llm_response(input_text, context=self.memory.short.search(...))
        self.memory.append("assistant", result)
        return result
```

**Step 2**：注册到消息总线（1 天）
```python

    def __init__(self, agent_id):
        ...
        self.bus = HaisClient(agent_id)
```

**Step 3**：加入 collaborate()（2 天）
```python
async def collaborate(self, mode, agents, payload):
    """替代 handoff() 的协作 API"""
    if mode == "chain":
        return await self._chain(agents, payload)
    elif mode == "gather":
```

### 5.2 渐进式方案：数据库选型路线

> 参考：8.8-hais-v0.2-design.md §6.1, §10

**不需要一上来就上 PostgreSQL 和 Redis**。按阶段逐步升级：

| 阶段 | 记忆 | 消息总线 | 复杂度 | 时间 |
|------|------|---------|--------|------|
| 🟢 **Phase 0** | 无记忆（v0.2.6） | HTTP handoff | 0 | 已就绪 |
| 🟡 **Phase 1** | SQLite（单文件） | 内存队列 | ⭐ | 1 周 |
| 🟠 **Phase 2** | SQLite + 本地向量 | HTTP 轮询 bus | ⭐⭐ | 1 周 |
| 🔴 **Phase 3** | PostgreSQL + pgvector | WebSocket bus | ⭐⭐⭐ | 2 周 |
| 🟣 **Phase 4** | PostgreSQL + Redis | WebSocket 集群 | ⭐⭐⭐⭐ | 2 周 |

**推荐起点**：Phase 1，一个下午就能跑通。

### 5.3 不需要立即做的

❌ 内存数据库集群（Redis Sentinel/Sentinel）
❌ 消息持久化（Kafka/RabbitMQ）
❌ 多机部署（K8s）
❌ 性能监控（Prometheus/Grafana）

这些是 Agent 数量 >50、日活 >1000 时才需要考虑。

## 6. 决策清单

> 以下归纳自 8.8-hais-v0.2-design.md §十六，附推荐方案

| # | 决策点 | 推荐方案 | 理由 |
|---|-------|---------|------|
| 1 | **持久化选型** | Phase 1 用 SQLite，Phase 3 升 PostgreSQL+pgvector | 渐进式，降低初始门槛 |
| 2 | **消息总线协议** | 开发用 HTTP 轮询，生产用 WebSocket | HTTP 调试方便，WebSocket 延迟低 |
| 3 | **加密方案** | HMAC-SHA256 签名（必选）+ Fernet 加密（可选） | 签名免费，加密有性能开销 |
| 4 | **API 守恒** | 6 内部 + 1 外部 = 7，不需要突破 | 守恒原则是 HAIS 的核心特征 |
| 5 | **实施时间** | Phase 1 一周，Phase 2 两周，Phase 3 四周 | 比设计文档的 7 周更灵活 |
| 6 | **协作模式优先级** | 先做顺序链 + 并行汇聚 | 覆盖 80% 需求，投票/共识可晚一步 |
| 7 | **身份认证** | 本地开发用 dev key，生产用 Ed25519 | 简化开发，生产安全 |
| 8 | **代码仓库** | `hais_memory.py` + `hais_bus.py` + `hais_orchestrator.py` | 三文件架构，清晰可维护 |
