## 0. 诚实声明

> **本仓库的核心数字 97.3% 目前是理论推导值，不是临床实测值。**
>
> 推导链完整、数学自洽：
> ```
> μ=10⁻⁹ → 三层纠错 → 正向发育投影 F(Genome)
>         → 贝叶斯反向推理 P(Genome|Body)
>         → 5 传感器互信息累加 ~2.32 bits
>         → 后验概率 ~97.3%
> ```
>
> 但支撑这条链的**五个输入参数**（5 个传感器的敏感性/特异性/互信息）目前是"基于文献估算"，**没有一份可追溯的临床标定报告**。
>
> 这不意味着 97.3% 是错的。这意味着它**还没有被真正的临床数据检验**。本文档的任务就是把这条路铺好。

## 1. 五个传感器的精确定义

**当前状态**：文档只给了传感器名称（舌诊/脉诊/耳诊/手骨诊/面诊），没有手术层面的操作定义。

**需要标定的目标**：对每个传感器，给出以下 6 项定义。

### 1.1 传感器 #1 · 舌诊（Lingual Holographic Sensor）

| 字段 | 当前状态 | 标定需求 |
|------|---------|---------|
| **观测目标** | "舌象" | 具体测量的客观指标：舌色 RGB 值/苔色/苔厚指数/舌体大小/裂纹/齿痕 |
| **信号采集** | 目视观察 | 标准化拍照条件（D65 光源、45° 照明、色卡校准） |
| **阳性标准** | 无精确定义 | 对每个可测量的指标，定义阈值（如：舌色红=色差ΔE>15 vs 健康基线） |
| **参考标准 (Gold Standard)** | 无 | 以什么作为疾病的真值？组织活检？血液标志物？影像学？ |
| **目标疾病** | 无精确定义 | 舌诊反映什么疾病？消化系统？代谢类？还是所有疾病？ |
| **敏感性/特异性** | ~0.75/0.85 | 待标定，无具体数据 |

> **标定优先级**：先选定 1-2 个"高信噪比"的疾病（如：2 型糖尿病 vs 健康对照），在受控条件下测量舌象 vs 血糖 HbA1c 的关联。

### 1.2 传感器 #2 · 脉诊（Pulse Holographic Sensor）

| 字段 | 当前状态 | 标定需求 |
|------|---------|---------|
| **观测目标** | "脉象" | 压力传感器阵列的时频域特征：脉率/脉幅/升斜/降斜/浮中沉深度 |
| **信号采集** | 手指触诊 | 标准化压力传感器（3×3 阵列，腕部桡动脉 1-3cm² 范围） |
| **阳性标准** | 无精确定义 | 从时频特征提取的分类边界 |
| **参考标准** | 无 | 心血管状态：血压（动态血压监测）、心输出量（超声）、动脉硬化（PWV） |
| **目标疾病** | 无精确定义 | 心血管类？还是全身性？ |
| **敏感性/特异性** | ~0.70/0.80 | 待标定 |

### 1.3 传感器 #3 · 耳诊（Auricular Holographic Sensor）

| 字段 | 当前状态 | 标定需求 |
|------|---------|---------|
| **观测目标** | "耳穴反应" | 耳廓特定穴位的电阻抗/压痛阈值/皮肤温度 |
| **信号采集** | 探针触诊 | 标准化电阻抗探头（固定压力 50g，接触面积 2mm²） |
| **阳性标准** | 无精确定义 | 电阻异常阈值（如：<50kΩ vs 健康对照侧 >200kΩ） |
| **参考标准** | 无 | 对应脏腑的临床检查（如：胃对应上消化道内镜） |
| **目标疾病** | 无精确定义 | 耳穴-内脏对应需要先验证 |
| **敏感性/特异性** | ~0.72/0.82 | 待标定 |

### 1.4 传感器 #4 · 手骨诊（Second Metacarpal Holographic Sensor）

| 字段 | 当前状态 | 标定需求 |
|------|---------|---------|
| **观测目标** | "第二掌骨节肢反应点" | 掌骨桡侧 7 穴位的压痛等级/皮下结节/组织张力 |
| **信号采集** | 手指按压 | 标准化压力探头（直径 3mm，压力 100g、200g、400g 分级） |
| **阳性标准** | 无精确定义 | VAS 疼痛评分 >3/10，或对比健侧有明显组织张力差异 |
| **参考标准** | 无 | 对应脏腑的临床检查 | 
| **目标疾病** | 无精确定义 | Zhang 1981 声称 7 穴位对应全身，但需要逐个验证 |
| **敏感性/特异性** | ~0.68/0.78 | 待标定—这是当前最低估值的传感器 |

| **参考标准** | 无 | 血液生化指标 + 全身健康状况评估 |
| **目标疾病** | 无精确定义 | 面色反映循环系统？内分泌？肾脏功能？ |
| **敏感性/特异性** | ~0.71/0.83 | 待标定 |

## 2. 先验的标定

**当前状态**: $p_0 = 5\%$，文档写"典型人群发病率"。

**问题**: 发病率是**疾病和人群特定**的。糖尿病在 60 岁以上人群的发病率 ~20%，在普通人群 ~10%，在青年人群 <2%。用一个固定的 5% 涵盖所有情况是没有意义的。

**标定方案**：先验应该是**条件先验**而非固定值。

### 2.1 先验的分层模型

```
先验概率 = P(疾病 | 年龄, 性别, 家族史, 生活方式, 地区, ...)
         ≠ 固定值 5%
```

先验的标定分为三级：

| 级别 | 数据来源 | 精度 | 说明 |
|------|---------|------|------|
| **L1: 人群统计先验** | CDC / WHO / 国家卫健委统计年鉴 | ±1-3% | 年龄-性别分层发病率 |
| **L2: 个体风险先验** | 问卷 + 基础体检 | ±0.5-1% | 加入 BMI / 吸烟 / 家族史等风险因子 |
| **L3: 动态先验** | 个人健康历史 + 前次诊断结果 | ±0.1-0.5% | 贝叶斯序贯更新（上次的后验=下次的先验） |

### 2.2 先验对 97.3% 的敏感性

从 8.5 仿真数据看，不同先验下的准确率：

| 先验 | 准确率 | 精确率 | 召回率 | 适用场景 |
|------|-------|-------|-------|---------|
| 1% | **99.0%** | 68.3% | 32.1% | 罕见病筛查 |
| **5%** | **97.3%** | **85.6%** | **55.2%** | **当前假设** |
| 10% | 95.8% | 72.4% | 73.5% | 高风险人群 |
| 20% | 92.1% | 65.8% | 85.3% | 门诊初筛 |
| 50% | 85.4% | 60.2% | 91.7% | 专科转诊 |

> **核心发现**：97.3% 在 5% 先验下成立。但同一个传感器组合在 50% 先验下（专科转诊场景）准确率降到 85.4%——**97.3% 不是普适数字，是先验条件的产物**。

## 3. 传感器独立性的实证检验

**当前状态**: 假设传感器间相关系数 ρ=0.0（完全独立）。

**问题**: 如果传感器之间不完全独立，实际的互信息累加值会小于 2.32 bits，导致后验精度低于 97.3%。

### 3.1 冗余度量

定义传感器 $i$ 和传感器 $j$ 之间的**信息冗余度**：

$$
\text{Redundancy}(i,j) = \frac{I(X_i; X_j)}{H(X_i)}
$$

其中 $I(X_i; X_j)$ 是两个传感器的互信息，$H(X_i)$ 是传感器 $i$ 的熵。

| 冗余度 | 含义 | 对总互信息的影响 |
|-------|------|----------------|
| <0.1 | 几乎独立 | 累加效应接近 $n \times I_i$ |
| 0.1-0.3 | 弱相关 | 累加效应损失 5-15% |
| 0.3-0.5 | 中等相关 | 累加效应损失 15-30% |
| 0.5-0.7 | 强相关 | 累加效应损失 30-50% |
| >0.7 | 几乎冗余 | 第二个传感器几乎没有新信息 |

### 3.2 预计的传感器间冗余度

基于中医临床经验，我们给出**假设性估计**（待实证）：

| 传感器对 | 预计冗余度 | 原因 | 验证方案 |
|---------|---------|------|---------|
| 舌诊-面诊 | 0.3-0.5 | 都反映气血状态，部分信号通路重叠 | 拍照分析互相关 |
| 脉诊-耳诊 | 0.1-0.3 | 心血管状态 vs 神经反射，关联较弱 | 同时测量后解相关 |
| 手骨诊-耳诊 | 0.05-0.2 | 不同节肢系统的信号，预计较独立 | 对比压痛等级 |
| 舌诊-脉诊 | 0.2-0.4 | 消化系统 vs 循环系统，中等关联 | 代谢参数交叉验证 |
| 面诊-耳诊 | 0.1-0.2 | 不同神经节段支配 | 各自免疫组化标记 |

> **对 97.3% 的潜在影响**：如果 5 传感器平均冗余度 0.25，有效互信息从 2.32 bits 降到约 1.74 bits，对应的后验精度从 97.3% 降到约 **94.8%**。差距约 2.5 个百分点——有意义但不算致命。这正是实证标定的价值所在。

## 5. 在数据到齐之前，我们能做什么？

临床数据不是明天就能到手的。在等待资金和伦理审批的同时，以下几件事可以现在就做：

### 5.1 敏感性分析：放宽所有假设，看 97.3% 能跌到多少

```python
# 最差情景的蒙特卡洛仿真
# 假设: 传感器间相关 ρ=0.3 + 传感器质量下界 + 先验更不利

worst_case_sensors = [
    (0.65, 0.75),  # 舌诊 降低 10%
    (0.60, 0.70),  # 脉诊 降低 14%
    (0.62, 0.72),  # 耳诊 降低 14%
    (0.58, 0.68),  # 手骨诊 降低 15%
    (0.61, 0.73),  # 面诊 降低 14%
]
```

从 8.5 仿真数据直接看：当传感器质量降到"较差"水平（sens ~0.5, spec ~0.7），准确率从 97.3% 降至 ~**91%**。

**97.3% 的置信区间**（基于参数不确定性传播）：

| 情景 | 估计精��� | 与当前差距 |
|------|---------|----------|
| 乐观（独立+高质量） | **98.5-99.2%** | +1.2-1.9% |
| 当前假设 | **97.3%** | — |
| 中等相关（ρ=0.2） | **95.8-96.5%** | -0.8-1.5% |
| 弱传感器+中等相关 | **92-94%** | -3-5% |
| 悲观（强相关+低质量） | **85-89%** | -8-12% |

> **这个表格应该被加粗放在 97.3% 旁边**，而不是藏在一个角落里。坦白不确定性比假装确定性更科学。

### 5.2 用公开数据集做初步验证

在等不到大笔资金做 Phase II/III 临床实验的情况下，可以利用**已有的公开健康数据集**做预标定：

| 数据集 | 内容 | 与传感器相关性 | 可自动化程度 |
|--------|------|--------------|------------|
| **MIMIC-III/IV** (MIT) | ICU 患者 + 生命体征 + 舌图像（一部分有） | 脉诊 ↔ 心率/血氧/血压 | 高 |
| **UK Biobank** | 50 万人 + 面部照片 | 面诊 ↔ 全身健康参数 | 中 |
| **TCGA** (癌症基因组图谱) | 临床数据 + 基因表达 | 手诊/耳诊 ↔ 特定癌症 | 低（无直接体表测量） |
| **CGM 数据集** (Dexcom/Freestyle) | 连续血糖监测 | 舌诊 ↔ 血糖波动 | 中 |
| **OCT 眼底数据集** | 视网膜成像 | 微循环系统 ↔ 心脑血管风险 | 低（仅限于眼部） |

> **目前就可行的最小方案**：用 UK Biobank 的面部照片+健康参数，初步标定"面诊"的信息量。不需要伦理审批（数据已公开），只需要下载和计算。这能给出第一个非拍脑门的互信息值。

### 5.3 计算公式的普适化

把当前固定的公式：

$$
P_{\text{posterior}} = \frac{1}{1 + (1-p_0)/p_0 \cdot 2^{-\sum I_i}}
$$

升级为**参数化版本**，显式包含传感器相关性 $\rho$ 和先验条件：

$$
P_{\text{posterior}}(D | \text{Sensors}) = \frac{1}{1 + \frac{1-p(D|C)}{p(D|C)} \cdot 2^{-(\sum I_i - \frac{1}{2}\rho^T \Sigma_{\text{red}} \rho)}}
$$

其中：
- $p(D|C)$ = 条件先验（疾病 D 在人群特征 C 下的发病率）
- $\sum I_i$ = 各传感器的边际互信息之和（标定值，非估算值）
- $\rho$ = 传感器间相关系数向量（实测值）
- $\Sigma_{\text{red}}$ = 传感器间冗余协方差矩阵

这样一来，公式的结果**不再是 97.3% 这个单一数字**，而是一个**动态精度函数**——取决于疾病、人群、传感器质量和相关性。

## 6. 结论：97.3% 的重新定位

经过上述分析，97.3% 应该被重新定位为：

> **"在 5% 先验、传感器独立、传感器质量参数为 (0.70-0.75, 0.78-0.85) 的条件下，理论推导的最优精度上限。"**

它**不是一个临床实证结果**，而是一个**可证伪的理论预测**——当且仅当上述所有条件在真实临床数据中被验证时，它才成立。

**对仓库的信用影响**：诚实坦承这一点不是弱点。恰恰相反——只有坦白说"我们目前是理论预测，还需要临床标定"，这个框架才值得被认真对待。

> **Popper 的证伪主义**：一个理论的科学价值不在于它有多确定，而在于它有多容易被证伪。97.3% 的当前最大优势恰恰是**它非常容易被证伪**——找个 500 人的临床实验就能让它闭嘴或站起来。
