## 摘要

前 6 篇建立了全息诊断的完整数学框架（贝叶斯→发育→BCI→融合→仿真→时序）。本文将其工程化：设计 5 种核心传感器的硬件方案，给出完整的信号链和噪声预算，并提出 HoloScan 2.0 的系统集成架构。核心结论：一个 **¥5,000 以内**的传感器阵列（掌诊摄像头 + 耳穴阻抗探头 + 舌诊高光谱相机 + 脉诊压阻阵列 + 面诊 RGB-D 相机）即可实现 8.5 中"良好"级别（sens ≥ 0.8, spec ≥ 0.9）的诊断性能。

## 2 单传感器设计方案

### 2.1 S1: 掌诊成像仪

**设计原理**: 掌部全息穴位在病理状态下会出现颜色（红/白/紫）和温度变化，可通过高分辨率彩色成像 + 偏振光消除表面反射来捕获。

```
信号链:
[LED 环形照明 (6500K)] → [偏振片] → [手掌]
  → [交叉偏振片] → [5MP CMOS] → [ISP] → [特征提取]
  
核心参数:
  ─ 分辨率: 2592×1944 (5MP), 足够分辨 2mm 穴位
  ─ 照明: 12 颗高显色 (CRI >95) 白光 LED, 色温 6500K
  ─ 偏振: 线偏振 + 交叉偏振, 消除皮肤表面反射
  ─ 帧率: 5 fps (连续 3 秒获取 15 帧, 用于运动补偿)
```

**特征提取流水线**:

```
ROI 定位 (掌骨解剖标志) → 颜色空间转换 (RGB→LAB)
  → 局部色差分析 (ΔE 相对于基线)
  → 纹理分析 (GLCM 对比度、均匀性)
  → 温度梯度 (从颜色间接推算)
  → 机器学习分类器 (XGBoost 或 MobileNet)
```

**噪声预算**:

| 噪声源 | 幅度 | 抑制措施 |
|-------|------|---------|
| 环境光变化 | ±10% | 封闭遮光罩 + LED 恒流驱动 |
| 皮肤出汗 | ΔE ±3 | 干燥处理 | 
| 运动模糊 | 5-10 像素 | 高速电子快门 (1/200s) |
| 传感器热噪声 | SNR 45 dB | 帧平均 (15帧) |

### 2.2 S2: 耳穴阻抗探针

**设计原理**: 穴位电阻在病理状态下降（炎症→血管扩张→组织液增加→阻抗降低），通过 4 电极恒流源精确测量。

```
信号链:
[恒流源 (10μA, 1kHz)] → [激励电极] → [耳穴皮肤]
  → [测量电极] → [仪表放大器 AD8221] → [2阶 Butterworth LPF 100Hz]
  → [ADC 16-bit] → [校准] → [阻抗值]

核心参数:
  ─ 激励: 10μA 正弦波 @ 1kHz (符合医疗安全 IEC 60601)
  ─ 电极: 银/氯化银 (Ag/AgCl) 导电橡胶, 直径 2mm
  ─ 测量: 4 电极法 (Kelvin), 消除接触阻抗
  ─ 范围: 10kΩ - 2MΩ, 分辨率 100Ω
  ─ 响应: 每次测量 < 200ms
```

**穴位定位**:
- 参考 WHO 90 标准耳穴定位图
- 用 3D 打印耳廓固定夹具确保重复定位
- 定位误差 < 1mm

### 2.3 S3: 舌诊高光谱仪

**设计原理**: 舌质的颜色（淡红/红/绛/紫/淡白）和舌苔的厚度/颜色是标准全息指标。传统 RGB 相机丢失光谱信息，改用 5 波段 LED 分时照明获取多光谱数据。

```
信号链:
[5 波段 LED 序列照明] → [舌面] → [单色 CMOS]
  → [波段重建] → [光谱特征提取]

LED 波段 (FWHM 20nm):
  470nm (蓝)  — 舌苔厚度/白苔
  530nm (绿)  — 舌质红/绛
  590nm (琥珀) — 舌质淡白
  660nm (红)  — 微循环/舌下静脉
  850nm (NIR) — 舌下静脉深度
  
重建: 5 波段 → 伪彩色输出 + 光谱特征向量 (5维)
```

**舌下静脉分析**:
- NIR 850nm 成像穿透舌黏膜，暴露舌下静脉
- 静脉直径、曲度、迂曲程度 → 血瘀严重度

### 2.4 S4: 脉诊阵列

**设计原理**: 28 脉象参数核心是腕部桡动脉的压力波形。单点传感器易受定位误差影响，使用 3×3 MEMS 压阻阵列实现空间覆盖。

```
信号链:
[3×3 MEMS 压阻阵列] → [惠斯通电桥] → [仪表放大器 INA333]
  → [3阶 Sallen-Key LPF 30Hz] → [Δ-Σ ADC 24-bit]
  → [数字滤波] → [波形分割] → [参数提取]

核心参数:
  ─ 阵列: 3×3, 间距 3mm, 覆盖 9×9mm
  ─ 量程: 0-200 mmHg (3-27 kPa)
  ─ 分辨率: 0.1 mmHg (12-bit 等效)
  ─ 带宽: DC - 30 Hz
  ─ 噪声: < 3 μVrms (0.01 mmHg 等效)
  ─ 采样率: 500 Hz/通道

3D 打印腕托:
  ─ 可调节压力 (通过微距螺丝)
  ─ 定位标记对齐桡骨茎突
  ─ 适配腕围 14-22cm
```

**波形分割算法**:

```
p(t) → 峰值检测 → 周期分割
  → 分类 (平/浮/沉/数/迟/弦/滑/涩/洪/细/微)
  → 参数提取 (k1, k2, tn, AW, PP, DR, KR)
```

### 2.5 S5: 面诊 RGB-D 相机

**设计原理**: 面部的五色（青/赤/黄/白/黑）对应五脏，结合 3D 结构光获取面部轮廓特征。

```
信号链:
[RGB 相机] + [结构光投影 (940nm)]
  → [RGB + 深度同步] → [面部关键点检测]
  → [颜色分析 (Dlib 68 点 ROI)]
  → [肤色均匀度、光泽度]

核心参数:
  ─ RGB: 1080p, 30fps
  ─ 深度: 640×480, 精度 1mm @ 50cm
  ─ 照明: 环形日光 (CRI >90)
  ─ ROI: 额 (心)、鼻 (脾)、左颊 (肝)、右颊 (肺)、下颌 (肾)
```

## 4 校准与质量控制

### 4.1 初始校准

每次开始使用前，运行 30 秒自动校准：

```
S1 掌诊: 采集白平衡参考 (标准白板) → 色彩校正矩阵
S2 耳穴: 开路/短路自检 → 参考电阻测量
S3 舌诊: 5 波段光谱参考 → LED 光功率归一化
S4 脉诊: 零压力基线 → MEMS 偏置校准
S5 面诊: 白平衡 + 深度校准
```

### 4.2 交叉验证

**每日金标准**: 使用模拟人体组织 phantom（明胶基，含模拟穴位）验证传感器一致性：

- 每日偏差 < 5% → 正常
- 每日偏差 5-10% → 自动修正
- 每日偏差 > 10% → 提示用户校准

## 5 成本-精度优化

### 5.1 投资优先级

基于 8.5 蒙特卡洛仿真结果（传感器质量对召回率的决定性影响）：

| 优先级 | 传感器 | 当前成本 | 升级投资 | 预期精度提升 |
|-------|-------|---------|---------|------------|
| 1 | **舌诊** | ¥600 | +¥400 (7波段→12波段) | 召回率 +8% |
| 2 | **脉诊** | ¥400 | +¥300 (3×3→5×5阵列) | 召回率 +5% |
| 3 | **耳穴** | ¥150 | +¥100 (单频→多频) | 召回率 +3% |
| 4 | **掌诊** | ¥300 | +¥200 (RGB→多光谱) | 召回率 +2% |
| 5 | **面诊** | ¥500 | +¥300 (更高分辨率) | 召回率 +1% |

**推荐**: 优先投资舌诊和脉诊，这两项对召回率影响最大。

### 5.2 降级模式

当某个传感器故障或信号质量差时，HoloScan 2.0 进入**降级模式**：

```
5 传感器全开 → 召回率 94.3%
缺少 1 个传感器 → 召回率 ~85-90%
缺少 2 个传感器 → 召回率 ~70-80%
仅掌诊 → 召回率 ~20-30% (单一穴位, 仅供快速筛查)
```
